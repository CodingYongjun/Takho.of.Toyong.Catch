## 프로젝트 소개
<p>
  jquery를 이용하여 만든 두더지게임 형식의 게임 입니다. 평소 좋아하는 게임인 동물의숲 게임의 캐릭터를 이용하여 만들었습니다.
  난이도 선택 및 결과의 따른 보상결과를 다르게 주었습니다.
</p>
